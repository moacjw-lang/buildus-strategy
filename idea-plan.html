02 포지셔닝 & 세대론
왜 빌드어스가 지금 필요한가. 시장의 구조적 공백.

인테리어 플랫폼 3세대

세대대표소비자에게 주는 것한계
1세대 셀인카페 "나는 이렇게 했어" (경험 공유) 파편화, 도구 없음, 느낌 수준
2세대 오늘의집 "이런 거 어때?" (영감 + 쇼핑) 시공 관리 없음, 감성 DNA
3세대 빌드어스 "너는 이렇게 해" (실행 도구 + 공정가) —
핵심 차이: 1세대는 읽고, 2세대는 구경하고, 3세대는 한다.
빌드어스에서 "하는 것" = 가이드 보면서 자재를 주문하고, 체크리스트 켜고 시공한다.
현재 정보 시장의 구조적 문제
채널문제
유튜브업체의 영업 도구. 핵심은 일부러 안 알려줌 (알려주면 자기한테 안 맡기니까)
셀인카페일반인의 "느낌" 공유. 구조화 안 됨, 도구 없음
쿠팡/네이버"뭘 사야 하는지 모르는" 사람에게 무력 (검색창에 뭘 쳐야 하는지도 모름)
빌드어스가 메우는 갭:
전문가의 진짜 현장 지식이 소비자에게 제대로 전달된 적이 없다.
업체는 일부러 안 알려주고, 일반인은 정확하지 않고, 카페는 구조화가 안 됨.
빌드어스가 이 갭을 메운다 — 월 2건 이상 현장을 돌리는 실무자의 정확한 지식으로.
리포지셔닝: "싸다" < "실패하지 않는다"
기존 메시지수정 메시지근거
"빌드어스에서 사면 싸다" "빌드어스에서 사면 실패하지 않는다" 가격은 쿠팡이 따라올 수 있지만, 전문성은 불가능 (ManoMano vs Amazon 교훈)

가격은 "덤"이지 "핵심"이 아닙니다. 메시지 순서: ① 뭘 사야 하는지 알려드립니다 → ② 실패 안 합니다 → ③ 그리고 업체 가격입니다.

비용 분석기 (자동견적 리포지셔닝)

"내 인테리어 예상 비용 분석" — 견적서가 아닌 비용 분석 리포트
WHAT — 무엇인가
  • 소개팅 앱 방식으로 공간별 이미지 선택 → 알고리즘이 자재 자동 배정 → 비용 분석 리포트 생성
  • 네이버 부동산 연동으로 아파트 면적 자동 산출
  • 가격은 범위로 표시: "예상 자재비: 280~350만원"
  • 세 가지 시나리오 제시: 절약형(셀프) / 표준형(반셀프) / 업체 의뢰
WHY — 왜 수정하는가
Angi의 교훈: $1.8B 매출 회사도 "고정가 모델"은 실패했다. 인테리어에서 "정확한 가격"을 약속하면, 현장 변수에 의해 반드시 어긋남 → 신뢰 파괴.
"견적서"라는 단어 자체가 정확도 기대를 높임. "비용 분석 리포트"로 바꾸면 범위 제시가 자연스러움.
HOW — 핵심 UX
[1단계] 이미지 선택 (소개팅 앱 방식) → 주방/거실/침실/욕실별 스타일 이미지 선택 → 예산 범위 설정 [2단계] AI가 자재 자동 배정 → 선택한 스타일 → 구체 자재 매핑 (m2당 단가) → 필요한 모든 공사 종류 + 방법 상세 나열 [3단계] 비용 분석 리포트 출력 → "업체에 맡기면 600만원. 빌드어스와 함께 하면 280만원." → "320만원 절약" — 이 숫자가 전면에 → 각 공사 항목에 자재 구매 링크 + 키트 구매 버튼 [4단계] 전환 → "이 분석에 사용된 자재, 빌드어스에서 바로 구매하세요" → 키트 / 개별 자재 장바구니
KPI
지표목표근거
리포트 생성 완료율70%+시작한 사람 중 결과까지 도달
리포트 → 자재 클릭30%+비용 분석 결과에서 자재 링크 클릭
리포트 → 자재 구매5~10%EnergySage 벤치마크 3~8%

자재 마켓플레이스

"전문가가 골라준 자재를, 전문가 가격에" — 쿠팡에서 살 수 없는 것
WHAT — 구조

  • 중량/부피 자재: 제조사/1차 유통사 입점 → 물류 리스크 0, 수수료 10%
  • 비취급 자재: 점진적 입점사 확대
WHY — 쿠팡과 왜 다른가
구분쿠팡빌드어스
가격소비자가 (인터넷가)업체가 (20~30% 저렴)

킬러 프로덕트 신규 Phase 1

공정별 자재 키트 (One-Box Kit)

"타일 시공에 필요한 모든 자재, 한 박스에" — 인테리어판 IKEA
리서치 전체에서 가장 중요한 발견. 한국에 경쟁자 0개. 유통사를 보유하므로 Day 1부터 제작 가능.
셀프빌더의 #1 불만 "뭘 사야 하는지 모르겠다"를 정면 해결.
WHAT — 초기 키트 5종
#키트명주요 구성품예상 가격
1방수 키트 (욕실/베란다)방수액, 부직포, 롤러, 실리콘건, 마스킹테이프39,000원~
2타일 시공 키트타일 본드, 줄눈, 십자, 스페이서 (도구 포함/미포함 옵션)49,000원~
3도장 키트페인트, 롤러세트, 마스킹, 양생비닐, 퍼티35,000원~
4마루 시공 키트바닥재, 방습필름, 몰딩, 실리콘면적별 상이
5줄눈 보수 키트줄눈제, 스크래퍼, 스펀지 (가장 쉬운 셀프 공정 → 입문용)15,000원~
WHY — 왜 킬러인가
  • 구매 결정 간소화: "뭘 사야 하지?" 고민 → "이 키트 하나면 됩니다" 0초 결정
  • 번들 마진: 개별 판매보다 마진율 높음 (물류 효율 + 심리적 "싸다" 인식)
  • 가이드 직결: 키트 박스에 QR코드 → 시공 가이드 영상 바로 접근
  • 유튜브 콘텐츠: "이 키트 하나로 욕실 방수 끝" = 강력한 콘텐츠 소재
  • 비교 불가: 쿠팡에 "방수 키트"를 검색해도 나오지 않음 — 이게 차별화
HOW — 마진 구조
개별 구매 시: 타일 본드 15,000 + 줄눈 8,000 + 십자 3,000 + ... = 45,000원 키트 구매 시: 39,000원 (13% 할인처럼 보임) 실제 마진: 키트 원가 ~22,000원 → 마진 17,000원 (43%) 개별 판매 시: 마진 합계 ~13,000원 (29%) → 키트 마진 > 개별 판매 마진 ✓
벤치마크
  • IKEA: 가구 조립 키트 → 15~20% 전환율
  • Home Depot Project Kit: DIY 프로젝트별 번들 → AOV 3~10배 상승
  • Wickes 영국: 공정별 자재 번들 → "여기서 다 살 수 있다" 인식 핵심
KPI
지표목표
가이드 조회 → 키트 구매 전환15~20%
키트 AOV35,000~50,000원
키트 재구매율 (다음 공정 키트)40%+

AI 구매 어시스턴트

"상담"이 아닌 "구매 안내" — 대화가 곧 장바구니
WHAT
  • 16개 공정 체크리스트(400+ 항목) 기반 RAG 챗봇
  • 모든 답변에 자재 추천 + 구매 링크 포함
  • "내 프로젝트에 저장" 기능 → 나중에 한번에 구매 가능
WHY — "상담"에서 "구매"로
AS-IS: "인테리어에 대해 무엇이든 물어보세요!" (상담) TO-BE: "어떤 공사를 하고 계신가요? 필요한 자재를 찾아드릴게요." (구매 안내) 대화 예시: Q: "욕실 방수 꼭 해야 하나요?" A: "네, 필수입니다. [체크리스트 기반 설명]..." 방수에 필요한 자재: - 방수액 (추천: OO제품, 12,000원) [구매하기] - 부직포 (추천: OO제품, 8,000원) [구매하기] → 또는 [방수 키트 한번에 39,000원] ← 킬러 전환
벤치마크
  • Amazon Rufus: AI 쇼핑 어시스턴트 → 대화형 상품 추천 + 바로 장바구니
  • Klarna AI: 월 250만 대화, 고객서비스 직원 700명 대체
비용 & KPI
항목수치

프로젝트 관리 도구 (무료)

"인테리어판 네비게이션" — 네비는 무료, 도착지에서 돈 번다
  • 체크리스트: 16개 공정별 체크리스트 기반. "다음에 뭐 해야 하지?"를 해결
  • 예산 트래커: 비용 분석 리포트 예산 vs 실제 지출 비교 대시보드
  • 사진 기록: 공정별 사진 촬영 → AI가 공정 태그 자동 분류 (UGC 인프라)
  • 자재 목록: "이 공정에 필요한 자재" 자동 생성 → 바로 구매
핵심 원칙: 이 도구 자체로 돈을 벌지 않음. 도구 안에서 자재를 사게 만드는 구조.
"타일 공정 시작" 체크 → "이 공정에 필요한 자재 리스트" 자동 → "빌드어스에서 바로 구매"

신규 Phase 1

업체 견적서 비교 분석기

TAM을 "셀프빌더"에서 "인테리어 전체 소비자"로 확장하는 도구

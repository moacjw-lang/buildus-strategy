아이디어 자체는 좋습니다. 해자도 있고, 방향도 맞습니다. 하지만 계획서의 수치 가정이 현실의 2~5배 과대평가되어 있고, 4명이 동시에 만들 수 있는 범위를 크게 초과합니다. 아이디어 등급 A-를 현실 등급 A-로 만들려면, 범위를 1/3로 줄이고 숫자를 현실에 맞춰야 합니다.

PART 1 숫자가 틀렸다 — 수치 보정

1. 구매 전환율 8%는 비현실적

치명적

이커머스 평균 구매 전환율은 1~3%입니다. 계획서의 8%는 이 수치의 3~8배.

계획서 가정
방문자 5,000 × 전환율 8% = 400건 구매
현실 보정
방문자 5,000 × 전환율 2~3% = 100~150건 구매

왜 8%가 안 되는가:

  • Wirecutter (최고의 가이드→커머스 모델)도 클릭률 3~7%, 그것도 아마존이라서 가능한 수치
  • 한국 소비자의 기본 행동: 가이드에서 제품명 확인 → 네이버 쇼핑에서 최저가 검색
  • 인테리어 자재는 범용 상품 — "OO브랜드 타일 본드"를 배우면 어디서든 살 수 있음
  • IKEA와 달리 구조적 잠금이 없음 (IKEA 부품은 IKEA에서만 구매 가능)
영향: 전환율이 8% → 3%로 내려가면, 같은 트래픽에서 매출이 62.5% 감소합니다. 수익 모델의 모든 계산이 달라집니다.

2. AOV 40만원은 3~4배 과대

치명적

셀프 인테리어는 공정별로 나눠서 진행합니다. 한 번에 40만원어치를 사는 게 아닙니다.

실제 구매 패턴예상 금액
"이번 주 욕실 방수" → 방수 키트39,000원
"다음 주 타일" → 타일 + 본드 + 줄눈15~20만원
공구류 (첫 구매 시)5~10만원
페인트 + 도장 용품5~8만원
계획서 가정
AOV 40만원
현실 보정
AOV 8~15만원 (공정별 분할 구매)
복합 영향 (전환율 + AOV 동시 보정):

계획서: 5,000 × 8% × 40만 = 월 GMV 1.6억 → 매출 3,200만원
현실: 5,000 × 3% × 12만 = 월 GMV 1,800만원 → 매출 360만원

9배 차이. 같은 유입이라도 실제 매출은 계획의 1/9 수준.

3. 6개월에 월 5,000명 유입은 안 된다

심각

SEO:

  • 새 도메인이 유의미한 검색 유입을 받기까지 최소 6~12개월
  • "인테리어 비용" 키워드는 네이버 블로그/집닥/오늘의집이 이미 점령
  • 아파트별 비용 데이터는 좋은 전략이지만, 한 사람의 추정치 vs 실제 통계 → 신뢰도 문제

유튜브:

  • 인테리어 유튜브 1,000구독 달성까지 평균 3~6개월
  • 유튜브 → 웹 전환율 0.1~0.9%
  • 구독자 1만 × 0.5% = 유튜브에서 월 50명 유입 — 의미 없는 수준

PART 2 구조적 문제 — 비즈니스 모델

4. "쿠팡은 경쟁자가 아니다"는 반만 맞다

심각

대표님 말이 맞는 부분: 시공업체는 쿠팡에서 안 삼.
하지만 빌드어스 타겟은 업체가 아니라 소비자(셀프/반셀프)입니다.

자재 유형소비자가 사는 곳빌드어스 우위
페인트, 마스킹, 실리콘, 기본 공구 쿠팡 (로켓배송 내일 도착) 없음
타일, 방수재, 전문 본드, 마루 네이버 스마트스토어 / 전문몰 있음 (업체가 수준)
대형 자재 (시멘트, 석고보드 등) 동네 건재상 / 유통사 제한적 (배송 문제)
정확한 표현: "쿠팡이 경쟁자가 아니다"가 아니라 "쿠팡이 약한 전문 자재 영역에서만 우위가 있다." 범용 자재에서는 쿠팡 로켓배송을 이길 수 없습니다. 그리고 범용 자재가 셀프빌더 구매의 30~40%를 차지합니다.

6. 구조적 잠금(Lock-in)이 없다

심각
플랫폼잠금 메커니즘전환 비용
IKEA IKEA 부품은 IKEA에서만 구매 가능 매우 높음
쿠팡 로켓와우 멤버십 (월정액 선결제) 높음
오늘의집 포인트 적립, 집들이 커뮤니티 중간
빌드어스 ??? 없음

빌드어스의 가치 = "뭘 사야 하는지 알려주고 + 싸게 팔고"

문제: 이 두 가치 모두 다른 곳에서 대체 가능합니다. 가이드를 다 봤으면 뭘 사야 하는지 알게 됨 → 어디서든 구매 가능.

유일하게 잠금이 작동하는 것: "키트"
이 조합을 다른 곳에서 직접 구성하기 귀찮음 → 키트에서 사는 게 편함.
키트가 킬러인 진짜 이유가 여기에 있습니다. 하지만 키트만으로 전체 사업을 지탱하기엔 SKU가 너무 제한적.

7. 실제 TAM이 생각보다 작다

주의
한국 연간 아파트 인테리어: 약 30만 건 셀프/반셀프 비율 (15~20%): 4.5~6만 건/년 온라인 플랫폼 사용 의향 (50%): 2.5~3만 건/년 빌드어스 시장 점유율 5% (매우 낙관적): 1,250~1,500건/년 = 월 100~125건 × AOV 12만원 = 월 GMV 1,200~1,500만원 × 마진 20% = 월 매출 240~300만원

"셀프빌더만 타겟"하면 실질 TAM이 사이드 프로젝트 수준입니다.
견적 비교 분석기로 일반 소비자(80%)까지 넓히는 것은 좋은 방향이지만, 그 전환율(일반 소비자 → 반셀프)은 미검증.

PART 3 실행 문제 — 만들 수 있는가

8. 4명이 Phase 1을 다 만들 수 없다

치명적

Phase 1에 나열된 10개 기능은 사실상 5~6개의 별도 프로덕트입니다.

기능실제 개발 범위개발자 1명 기준
비용 분석기 이미지 선택 UI + 자재 매핑 알고리즘 + 리포트 생성 6~10주
이커머스 마켓플레이스 상품 카탈로그, 결제, 배송 연동, 주문관리, 환불 12~20주
AI 챗봇 RAG 파이프라인, 프롬프트 튜닝, 자재 연결 UI 3~5주
프로젝트 관리 체크리스트, 예산 트래커, 사진 업로드, 대시보드 6~10주
견적 비교 분석기 항목 입력 UI, 비교 로직, 결과 리포트 3~5주
아파트별 비용 DB + SEO 데이터 수집, CMS, 랜딩 페이지 SEO 최적화 2~4주

합계: 32~54주 (8~13개월) — 개발자 1명이 다른 일 안 하고 풀타임 개발만 할 때.

6개월에 가능한 것 (현실적):
비용 분석기 + 키트 쇼핑몰(카페24/Shopify 기반) + AI 챗봇 MVP.
이것도 빡빡합니다.

9. 키트 실행 복잡도를 과소평가

주의

"방수 키트 39,000원" — 깔끔해 보이지만:

  • 욕실 크기마다 방수액 양이 다름 → 사이즈별 SKU 필요 (1평/2평/3평용)
  • 키트 안에 하나라도 불량이면 전체 반품 → 반품 비용 3~5배
  • 구성품 하나가 품절 → 키트 전체 판매 중지
  • "도구 포함/미포함" 옵션만 해도 SKU 2배
키트 5종 × 사이즈 3종 × 도구옵션 2종 = 30개 SKU 각 SKU마다: - 구성품 재고 동기화 - 패킹 작업 - 품질 검수 - 반품/교환 처리 = 한 명의 풀타임 업무

10. CS 인프라가 완전히 빠져 있다

심각

이커머스에서 CS는 "부가 기능"이 아니라 핵심 운영입니다.

  • "이 자재 우리 집에 맞나요?" "배송 언제 오나요?" "색상이 다릅니다"
  • 건축자재 반품은 무겁고 비쌈 (30kg+ 택배 불가한 것도 있음)
  • 월 100건 주문 → CS 문의 20~40건 (이커머스 평균 문의율 20~40%)
  • 시공 관련 문의가 들어오면? "이거 어떻게 시공해요?" — 답변에 시간 소요 큼
계획서에 CS가 한 줄도 없습니다.
이커머스를 해본 사람이라면 "CS 때문에 못 하겠다"가 가장 먼저 나올 이야기입니다. 4명 팀에서 CS 전담은 불가능합니다.
PART 4 그래도 되는 이유 — 진짜 강점

비판 속에서도 흔들리지 않는 것들

강점
강점왜 진짜인가비판으로도 안 깨지는 이유
유통사 보유 자재 원가/품질/호환성을 누구보다 정확히 앎 이건 돈/시간으로 따라잡을 수 없는 해자
키트 컨셉 한국에 "공정별 자재 키트"를 파는 곳이 0개 수치 가정이 틀려도, 이 상품 자체의 가치는 유효
체크리스트 16개 400+ 항목의 현장 지식 데이터베이스 AI 챗봇의 지식 기반으로 고유한 가치 제공
업계 내부자 포지션 인테리어 업체 + 유통사 동시 운영 가이드/키트/가격의 세 가지 모두의 소스
"실패하지 않는다" 포지셔닝 가격이 아닌 전문성으로 차별화 쿠팡이 따라올 수 없는 방향이 맞음
핵심: 아이디어와 해자는 진짜입니다. 문제는 아이디어가 아니라 실행 범위와 수치 가정입니다. 범위를 줄이고 숫자를 현실에 맞추면, 이건 됩니다.

핵심 발견

빌드어스의 각 요소는 해외에서 개별적으로 검증됨. 하지만 이 요소들을 한 플랫폼에서 결합한 곳은 아직 없음. 이것이 기회이자 리스크 — Houzz처럼 "다 하다가 아무것도 못할" 수 있으므로 Phase별 집중이 필수.

반드시 피해야 할 함정

Houzz의 교훈: 영감(사진)으로 사람을 모으고 구매를 기대하면 실패. "구경하는 사람 ≠ 사는 사람"
Angi의 교훈: 인테리어에서 고정가/표준화를 섣불리 시도하면 실패. 현장 변수가 너무 많음.
Bunnings UK의 교훈: DIY 문화가 없는 곳에서 순수 셀프만 타겟하면 실패. 반셀프/직영공사까지 포함해야.

핵심 인사이트

"읽기만 하는 콘텐츠"보다 "도구를 사용하는 콘텐츠"의 전환율이 3~10배 높다.
빌드어스는 가이드(읽기) + 자동견적/체크리스트(도구)를 결합하므로, 5~15% 전환율이 현실적 목표.

기존 04_사업성-분석의 전환율(0.75%)은 "도구 없이 콘텐츠만 있을 때"의 수치. 도구가 제대로 작동하면 1.5~3%까지 가능 → 12개월 시점 월 매출 1,150만 → 1,700~3,900만원으로 상향.

IKEA 플래너가 가장 가까운 벤치마크

IKEA 온라인 플래너 사용자의 구매 전환율: 15~20%. 플래너로 설계한 주문의 평균 객단가가 일반의 3~5배.

빌드어스 자동견적도 동일한 구조: "도구로 만들어본 결과물 → 필요한 자재가 있다 → 바로 구매"

A-1. 공정별 자재 키트 (One-Box Kit) Phase 1

"타일 시공에 필요한 모든 것, 한 박스" — 빌드어스의 킬러 상품

벤치마크

Home Depot "Project Kit", Wickes 공정별 번들, IKEA 조립 키트

왜 킬러인가

1. "뭘 사야 하는지 모르겠다" = 셀프빌더 #1 불만을 완벽히 해결
2. 번들 판매는 개별 판매보다 마진이 더 좋음
3. 키트 박스에 빌드어스 QR코드 → 가이드 바로 접근
4. 유통사를 이미 보유 → Day 1부터 키트 제작 가능
5. 한국에 인테리어 공정별 키트를 파는 곳이 없음

초기 5종

방수 키트 / 타일 시공 키트 / 도장 키트 / 마루 시공 키트 / 줄눈 보수 키트

빼야 할 아이디어 3개

B-1. 소비자용 시공 상세도면 → 축소

왜 빼야 하는가

1. 스케일 불가 — 아파트 평형별, 구조별 맞춤 도면은 4인 팀으로 불가능

3. 반셀프빌더에게 필요한 건 도면이 아니라 체크리스트 + 가이드 + 자재 목록

대안

"공정별 표준 시공 가이드 일러스트" — 한번 만들면 모든 사용자가 사용, 스케일 가능

B-2. 중고자재몰 → Phase 3 이관

왜 빼야 하는가

1. 양면 마켓 콜드스타트가 가장 어려운 유형
2. 자재가 무겁고 큼 → 택배 불가 → 지역 직거래만 가능 → 매칭 확률 극히 낮음
3. 전 세계적으로 중고 건축자재 마켓플레이스로 의미 있게 성공한 사례 없음

대안

커뮤니티 게시판의 "자재 나눔/거래" 카테고리로 운영 (개발 비용 0)

B-3. 대규모 커뮤니티 → Phase 1 최소화

왜 빼야 하는가

1. 빈 커뮤니티 = 유령도시. Day 1에 열어봤자 아무도 안 옴
2. 커뮤니티 운영은 숨은 리소스 블랙홀 (스팸, 분쟁, 품질 관리)
3. Phase 1의 핵심은 "도구 → 구매" 전환 검증이지, 커뮤니티 구축이 아님

대안

Phase 1: AI 챗봇이 Q&A 80% 처리 + 카카오 오픈채팅방 (개발 비용 0)
